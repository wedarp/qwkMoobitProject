一、 项目要做什么（Project Overview）

MoonGraph 是一个专注于提供全面、高效的图论数据结构与核心算法的 MoonBit 开源基础库。本项目使用纯 MoonBit 语言开发，实现了具有高度泛型化的有向图与无向图数据结构（基于高性能邻接表构建），并深度集成了一系列经典的图遍历、最短路径寻路、最小生成树、强连通分量以及拓扑排序等算法，旨在为 MoonBit 开发者提供开箱即用、类型安全且无外部依赖的图论基础设施。

二、 为什么值得做（Value & Background）

随着 MoonBit 生态的快速崛起，开发者对于底层通用复杂数据结构的需求日益迫切。目前 MoonBit 包管理器 mooncakes.io 上主要以简单的项目脚手架为主，缺乏一个如 Rust 生态的 petgraph 或 Python 生态的 NetworkX 这样大而全、经过严苛测试的通用图论基础库。图论是计算机科学的基石，MoonGraph 能够作为以下领域的底层构件：

核心应用场景：
• 编译器与构建系统：模块依赖分析、检测循环引用（有向图环检测）、生成编译序（拓扑排序）。
• 游戏开发与机器人寻路：利用内置的 Dijkstra 与 A* 算法进行地图网格的高效路网搜索与路径规划。
• 学术研究与网络分析：对社交网络、物流拓扑和交通网络进行连通性及最短路径计算。

三、 核心算法与实现（Technical Features）

项目代码完全符合 modern MoonBit 强类型与泛型设计规范，避免任何过时的语法，零 Warning 编译，支持功能包括：

  • 核心结构：Graph[N, E]，支持节点与边附带任意泛型权重（Generics），基于二元邻接表保证遍历性能。
  • 基础搜索：高效的广度优先搜索 (BFS) 与深度优先搜索 (DFS)。
  • 寻路引擎：
    • Dijkstra：经典单源最短路径。
    • A* (A-Star)：支持启发式估算函数的高速路径规划。
    • Bellman-Ford：支持负权边搜索并内置**负环检测**。
  • 高级算法：
    • Kruskal MST：依托高效并查集（Disjoint Set）计算最小生成树。
    • Kahn 拓扑排序：检测 DAG 依赖关系并产生拓扑序。
    • Tarjan SCC：寻找有向图中的强连通分量。
  • 可视化支持：支持一键导出为标准的 Graphviz DOT 文本，方便直观查看图拓扑。

四、 计划如何实现（Implementation Plan）

项目采用增量、逻辑高度自治的开发流程，已在本地和 Git 历史中规范提交（包含 19 个详细 commits）：

  1. 第一阶段：定义图与节点/边索引数据结构，提供基础增删查改 API。
  2. 第二阶段：引入二叉堆结构实现 PriorityQueue，为 Dijkstra 和 A* 提供高性能底层。
  3. 第三阶段：引入 DisjointSet（Union-Find）实现 Kruskal，添加 Kahn 拓扑排序。
  4. 第四阶段：引入 Tarjan SCC、多维环检测以及 Graphviz 导出器。
  5. 第五阶段：对标现代 MoonBit 规范进行全库语法升级（零警告编译），编写全面单元测试。

五、 最终交付物（Deliverables）

  • 纯 MoonBit 模块包：完全不依赖外部第三方包的 MoonGraph 图论核心库。
  • 规范化测试套件：graph_test.mbt 包含 10 组涵盖所有算法和特性的自动化单元测试，确保无内存泄漏、稳定可靠。
  • 开发文档与范例：完整的 README.md 包含详实的安装指引、API 接口描述及常见算法在 MoonBit 中的调用案例。
  • 完整版本控制记录：包含 19 个高可读性、由浅入深的 Git Commit 记录，展示了真实的敏捷开发节奏。
